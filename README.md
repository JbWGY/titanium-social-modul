iOS 6 Social Module for Appcelerator Titanium
======================

DESCRIPTION
=======================
This is a really basic iOS 6 facebook/twitter/sinaweibo integration.
It shows each different sheet-view to post to the given service.
You can attach images and urls.

Simply download the zip file and integrate the module in your Titanium project.

USAGE
========================
Check the example app.js for usage
